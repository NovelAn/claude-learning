# Claude Code 专项学习规划

> **学习目标**: 从零开始，系统掌握 Claude Code CLI 工具，成为 AI 辅助开发专家

**学习路径**: 基础 → Skills → Agents → MCP → 工作流 → 高级主题

**预计时间**: 6-12 个月（根据个人基础和投入时间）

**学习方式**: 理论学习 + 实战案例 + 个人项目

---

## 📖 学习指南简介

这是一个专门针对 **Claude Code CLI 工具**的系统性学习规划。Claude Code 是 Anthropic 官方推出的命令行工具，它将强大的 Claude AI 能力直接集成到开发工作流中。

通过本学习路径，你将掌握：
- ✅ Claude Code 的核心功能和最佳实践
- ✅ 创建自定义 Skills 来自动化你的工作流
- ✅ 开发 AI Agents 来处理复杂任务
- ✅ 集成 MCP Servers 来扩展能力边界
- ✅ 构建端到端的自动化工作流系统

### 为什么学习 Claude Code？

1. **提高开发效率** - 自动化重复性任务，专注创造性工作
2. **降低学习成本** - AI 辅助学习新技术栈和框架
3. **减少认知负担** - 让 AI 处理繁琐的细节，你关注架构和逻辑
4. **保持技术前沿** - 掌握未来编程的核心技能
5. **社区资源丰富** - 活跃的社区和丰富的第三方扩展

## 🚀 快速开始

### 第一步：安装 Claude Code

```bash
# 使用 npm 安装
npm install -g @anthropic-ai/claude-code

# 或使用 Homebrew (macOS)
brew install claude-code

# 验证安装
claude --version
```

### 第二步：配置环境

```bash
# 登录 Claude
claude login

# 查看帮助
claude --help

# 或在交互模式中
/help
```

### 第三步：开始学习

1. 📖 阅读 [`CLAUDE.md`](./CLAUDE.md) - 完整的学习指南
2. 📊 查看 [`progress/`](./progress/) - 追踪学习进度
3. 🎓 进入 [`modules/01-foundation/`](./modules/01-foundation/) - 开始第一阶段
4. 📝 完成实战案例并记录到 [`sessions/`](./sessions/)

## ❓ 常见问题

### Q: 我没有编程基础，可以学习吗？
**A**: 建议先学习基础的编程知识（推荐 Python 或 JavaScript），然后再开始本学习路径。

### Q: 学习需要多长时间？
**A**: 根据个人基础和投入时间：
- 每周 10 小时：6-12 个月
- 每周 20 小时：3-6 个月
- 全职学习：2-3 个月

### Q: 可以跳过某些阶段吗？
**A**: 不建议。每个阶段都建立在前一阶段的基础上。如果你已经掌握某些内容，可以快速复习。

### Q: 遇到问题怎么办？
**A**:
1. 查阅官方文档
2. 查看 GitHub Issues
3. 在社区提问
4. 记录问题并尝试自己解决

### Q: 学习完成后能达到什么水平？
**A**: 你将能够：
- 熟练使用 Claude Code 进行 AI 辅助开发
- 开发自定义的 Skills、Agents 和 MCP Servers
- 构建复杂的自动化工作流系统
- 为社区贡献你的开发成果
