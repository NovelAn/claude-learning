# 阶段二：Skills 开发与定制

在这个阶段，你将学习如何创建自定义 Skills，让 Claude Code 更好地服务于你的特定工作流程。

## 学习目标

完成本阶段后，你将能够：
- ✅ 深度定制内置 Skills
- ✅ 使用 Claude Agent SDK 开发自定义 Skills
- ✅ 打包和分发 Skills
- ✅ 构建个人专属的工具集

## 模块概览

### 模块 2.1 - 内置 Skills 深度应用
- `/commit` 的高级配置
- `/review-pr` 的定制化
- Hooks 系统的实现
- **实战案例**:
  - 配置符合团队规范的提交信息格式
  - 创建自动化代码审查工作流
  - 使用 Hooks 实现 pre-commit 自动化

### 模块 2.2 - 自定义 Skills 创建
- Skill 的基本结构
- Claude Agent SDK 使用
- 参数处理与用户交互
- 错误处理和日志
- **实战案例**:
  - 创建 `/deploy` Skill 实现一键部署
  - 创建 `/test-gen` Skill 自动生成单元测试
  - 创建 `/doc-gen` Skill 生成 API 文档
  - 创建 `/migrate` Skill 辅助数据库迁移

### 模块 2.3 - Skills Marketplace 与分享
- 打包和分发策略
- 版本管理
- 社区贡献
- **实战案例**:
  - 发布 Skill 到 npm/GitHub
  - 为团队创建私有 Skills 仓库

## 预计学习时间
- **总时长**: 4-6 周
- **每周投入**: 10-15 小时

## 前置要求
- ✅ 完成阶段一
- ✅ 熟悉 Node.js 或 Python
- ✅ 了解 CLI 工具开发基础
