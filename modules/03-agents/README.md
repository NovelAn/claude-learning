# 阶段三：Agents 架构与开发

深入学习 AI Agents 的设计、开发和部署，构建能够自主完成复杂任务的智能系统。

## 学习目标

完成本阶段后，你将能够：
- ✅ 理解 Agent 的核心概念和设计模式
- ✅ 使用 Claude Agent SDK 开发 Agents
- ✅ 构建 Multi-Agent 协作系统
- ✅ 实现 Agent 的记忆和状态管理

## 模块概览

### 模块 3.1 - Agent 基础概念
- Agent vs Script 的本质区别
- Claude Agent SDK 核心组件
- Agent 生命周期管理
- **实战案例**:
  - 构建自动日志分析器
  - 创建文件监控 Agent 处理 CSV 数据

### 模块 3.2 - Agent 设计模式
- 单一职责 vs 通用 Agent
- Agent Chains（链式调用）
- Multi-Agent Systems
- Agent 记忆与上下文管理
- **实战案例**:
  - 构建代码审查 Agent 链
  - 创建多 Agent 系统实现自动化测试
  - 实现 Agent 协作完成复杂重构

### 模块 3.3 - 高级 Agent 特性
- 决策机制与逻辑分支
- 错误恢复与重试策略
- 状态持久化
- 性能优化
- **实战案例**:
  - 实现具有状态管理的长期运行 Agent
  - 构建容错的分布式 Agent 系统
  - 优化 Agent 性能减少 Token 消耗

## 预计学习时间
- **总时长**: 6-8 周
- **每周投入**: 15-20 小时

## 前置要求
- ✅ 完成阶段一和阶段二
- ✅ 熟悉异步编程概念
- ✅ 了解基本的系统设计原则
